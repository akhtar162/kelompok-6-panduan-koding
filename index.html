Sejarah Pemrograman

Perjalanan panjang evolusi pemrograman komputer dari konsep awal hingga era modern.

Abad ke-19 Awal Konsep

Charles Babbage (1822 dan 1837)

Menciptakan Mesin Hitung Diferensial (Difference Engine) dan kemudian Mesin Analitik (Analytical Engine). Mesin Analitik ini dianggap sebagai prototipe komputer modern pertama karena dirancang untuk diprogram menggunakan kartu berlubang.

Ada Lovelace (sekitar 1843)

Dianggap sebagai programer komputer pertama di dunia. Ia menulis algoritma yang dimaksudkan untuk diproses oleh Mesin Analitik Charles Babbage, yang pada saat itu belum selesai dibangun. Algoritma ini dirancang untuk menghitung urutan bilangan Bernoulli.

1940-1950an Era Awal Komputer

Mesin Kode Awal

Komputer elektronik pertama, seperti ENIAC (1940-an), awalnya diprogram dengan mengatur ulang kabel fisik (wired control panel).

Konrad Zuse (1942–1945)

Menciptakan Plankalkül, yang dianggap sebagai bahasa pemrograman tingkat tinggi pertama, meskipun tidak diimplementasikan saat itu.

John von Neumann

Memperkenalkan konsep arsitektur komputer (arsitektur von Neumann) yang memungkinkan program disimpan di memori, memisahkan perangkat keras dan perangkat lunak.

Bahasa Tingkat Rendah

Assembly Language (1949)

Bahasa yang lebih sederhana daripada kode mesin (binary) yang menggunakan singkatan untuk instruksi.

Short Code (1949)

Salah satu bahasa tingkat tinggi pertama yang dikembangkan untuk komputer elektronik, meskipun harus diterjemahkan (diinterpretasi) setiap kali dijalankan.

Munculnya Compiler

Grace Hopper (1951)

Menciptakan compiler pertama (A-0 System), sebuah program yang menerjemahkan kode sumber (source code) menjadi kode mesin, mempercepat proses pemrograman.

Bahasa Tingkat Tinggi Pertama yang Digunakan secara Luas

Autocode (1952) oleh Alick Glennie

Bahasa pemrograman terkompilasi pertama yang digunakan.

FORTRAN (FORmula TRANslation) (1957)

Oleh tim yang dipimpin John Backus di IBM: Dianggap sebagai bahasa pemrograman tingkat tinggi pertama yang berfungsi dan digunakan secara luas, khususnya untuk perhitungan ilmiah dan matematika.

1960-Sekarang Perkembangan Modern

1960-an

Lahir bahasa seperti ALGOL (1960), LISP (1958, masih digunakan), COBOL (1959), dan BASIC (1964).

1970-an

C Language (1972) oleh Dennis Ritchie

Bahasa tingkat tinggi yang sangat berpengaruh, membentuk dasar bagi banyak bahasa modern lainnya.

SQL (Structured Query Language) (1974)

Bahasa standar untuk mengelola data dalam database relasional.

Notepad: Struktur HTML

Membangun kerangka dasar aplikasi notepad dengan elemen form dan wadah untuk catatan.

Kode Lengkap: index.html

Salin kode berikut dan simpan dalam file bernama index.html. Struktur ini mencakup header, formulir untuk membuat catatan baru, dan area untuk menampilkan daftar catatan.

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Aplikasi Notepad</title>
    <!-- CSS akan ditautkan di sini -->
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="container">
        <header>
            <h1>Aplikasi Notepad Sederhana</h1>
        </header>
        
        <main>
            <section class="note-form">
                <h2>Buat Catatan Baru</h2>
                <!-- Form untuk input judul dan isi catatan -->
                <form id="note-form">
                    <div class="form-group">
                        <label for="note-title">Judul Catatan</label>
                        <input type="text" id="note-title" placeholder="Masukkan judul catatan" required>
                    </div>
                    <div class="form-group">
                        <label for="note-content">Isi Catatan</label>
                        <textarea id="note-content" placeholder="Tulis catatan Anda di sini" required></textarea>
                    </div>
                    <button type="submit" class="btn">Simpan Catatan</button>
                </form>
            </section>
            
            <section class="notes-list">
                <h2>Daftar Catatan</h2>
                <!-- Wadah untuk menampilkan catatan yang sudah dibuat -->
                <div id="notes-container">
                    <p class="empty-state">Belum ada catatan. Buat catatan baru!</p>
                </div>
            </section>
        </main>
    </div>
    
    <!-- JavaScript akan ditautkan di sini -->
    <script src="script.js"></script>
</body>
</html>

Penjelasan Elemen Penting:

  • <form id="note-form">: Formulir yang akan menangani input pengguna. Atribut id penting agar bisa diakses oleh JavaScript.
  • <label for="...">: Label untuk meningkatkan aksesibilitas dan keterhubungan dengan input.
  • <input type="text" id="note-title">: Input teks satu baris untuk judul. id digunakan JavaScript untuk mengambil nilainya.
  • <textarea id="note-content">: Area teks multi-baris untuk isi catatan.
  • <button type="submit">: Tombol untuk mengirimkan formulir.
  • <div id="notes-container">: Wadah kosong yang akan diisi oleh catatan-catatan secara dinamis menggunakan JavaScript.

Notepad: Logika JavaScript

Menghidupkan aplikasi notepad dengan fungsi untuk menambah, menampilkan, dan menghapus catatan, serta menyimpannya di browser.

Kode Lengkap: script.js

Salin kode berikut dan simpan dalam file bernama script.js. Kode ini mengatur semua interaksi pengguna, mulai dari menyimpan catatan hingga menampilkannya di layar.

// Tunggu hingga seluruh konten DOM dimuat
document.addEventListener('DOMContentLoaded', () => {
    // Ambil elemen-elemen yang dibutuhkan dari DOM
    const noteForm = document.getElementById('note-form');
    const noteTitleInput = document.getElementById('note-title');
    const noteContentInput = document.getElementById('note-content');
    const notesContainer = document.getElementById('notes-container');

    // Muat catatan dari localStorage saat halaman dimuat
    let notes = JSON.parse(localStorage.getItem('notes')) || [];
    renderNotes();

    // Tambahkan event listener untuk submit form
    noteForm.addEventListener('submit', (e) => {
        e.preventDefault(); // Cegah form dari reload halaman

        // Ambil nilai dari input
        const title = noteTitleInput.value.trim();
        const content = noteContentInput.value.trim();

        // Validasi sederhana
        if (title === '' || content === '') {
            alert('Judul dan isi catatan tidak boleh kosong!');
            return;
        }

        // Buat objek catatan baru
        const newNote = {
            id: Date.now(), // Gunakan timestamp sebagai ID unik
            title: title,
            content: content,
            createdAt: new Date().toISOString()
        };

        // Tambahkan catatan baru ke array
        notes.unshift(newNote); // Gunakan unshift agar catatan baru muncul di atas

        // Simpan array catatan ke localStorage
        saveNotes();

        // Kosongkan form
        noteForm.reset();

        // Render ulang daftar catatan
        renderNotes();
    });

    // Fungsi untuk merender daftar catatan ke DOM
    function renderNotes() {
        // Kosongkan container terlebih dahulu
        notesContainer.innerHTML = '';

        if (notes.length === 0) {
            notesContainer.innerHTML = '
Belum ada catatan. Buat catatan baru!
';
            return;
        }

        // Loop melalui setiap catatan dan buat elemennya
        notes.forEach(note => {
            const noteElement = createNoteElement(note);
            notesContainer.appendChild(noteElement);
        });
    }

    // Fungsi untuk membuat elemen HTML untuk satu catatan
    function createNoteElement(note) {
        const noteDiv = document.createElement('div');
        noteDiv.classList.add('note-item');
        noteDiv.dataset.noteId = note.id; // Simpan ID di dataset untuk akses mudah

        noteDiv.innerHTML = `
            
${escapeHtml(note.title)}

            
${escapeHtml(note.content)}

            

                
            

        `;

        return noteDiv;
    }

    // Fungsi untuk menghapus catatan
    window.deleteNote = (id) => {
        if (confirm('Apakah Anda yakin ingin menghapus catatan ini?')) {
            // Filter array untuk menghapus catatan dengan ID yang cocok
            notes = notes.filter(note => note.id !== id);
            
            // Simpan perubahan dan render ulang
            saveNotes();
            renderNotes();
        }
    };

    // Fungsi untuk menyimpan catatan ke localStorage
    function saveNotes() {
        localStorage.setItem('notes', JSON.stringify(notes));
    }

    // Fungsi untuk mencegah serangan XSS (sederhana)
    function escapeHtml(text) {
        const map = {
            '&': '&',
            '<': '<',
            '>': '>',
            '"': '"',
            "'": '''
        };
        return text.replace(/[&<>"']/g, m => map[m]);
    }
});

Penjelasan Logika Penting:

  • localStorage: Fitur browser yang memungkinkan kita menyimpan data secara lokal (di komputer pengguna) bahkan setelah browser ditutup. Data ini akan bertahan.
  • JSON.parse() & JSON.stringify(): Karena localStorage hanya bisa menyimpan string, kita gunakan JSON untuk mengubah array objek catatan menjadi string (saat menyimpan) dan sebaliknya (saat mengambil).
  • DOMContentLoaded: Event ini memastikan bahwa kode JavaScript kita hanya berjalan setelah seluruh struktur HTML halaman telah dimuat dan siap dimanipulasi.
  • e.preventDefault(): Mencegah perilaku default dari form submit, yaitu me-reload halaman.
  • Date.now(): Digunakan untuk membuat ID unik untuk setiap catatan baru.
  • escapeHtml(): Fungsi keamanan penting untuk mencegah serangan Cross-Site Scripting (XSS) dengan mengubah karakter khusus HTML menjadi entitas HTML.

Salin kode berikut dan simpan dalam file bernama script.js. Kode ini akan mengelola semua logika aplikasi to-do list.

document.addEventListener('DOMContentLoaded', () => {
    const taskForm = document.getElementById('task-form');
    const taskInput = document.getElementById('task-input');
    const tasksContainer = document.getElementById('tasks-container');

    let tasks = JSON.parse(localStorage.getItem('tasks')) || [];
    renderTasks();

    taskForm.addEventListener('submit', (e) => {
        e.preventDefault();
        const taskText = taskInput.value.trim();

        if (taskText === '') {
            alert('Tugas tidak boleh kosong!');
            return;
        }

        const newTask = {
            id: Date.now(),
            text: taskText,
            completed: false
        };

        tasks.push(newTask);
        saveTasks();
        renderTasks();
        taskInput.value = '';
    });

    function renderTasks() {
        tasksContainer.innerHTML = '';
        tasks.forEach(task => {
            const taskElement = createTaskElement(task);
            tasksContainer.appendChild(taskElement);
        });
    }

    function createTaskElement(task) {
        const li = document.createElement('li');
        li.classList.add('task-item');
        if (task.completed) {
            li.classList.add('completed');
        }

        const checkbox = document.createElement('input');
        checkbox.type = 'checkbox';
        checkbox.checked = task.completed;
        checkbox.addEventListener('change', () => toggleTask(task.id));

        const label = document.createElement('label');
        label.textContent = task.text;

        const deleteBtn = document.createElement('button');
        deleteBtn.classList.add('delete-btn');
        deleteBtn.innerHTML = '×';
        deleteBtn.addEventListener('click', () => deleteTask(task.id));

        li.appendChild(checkbox);
        li.appendChild(label);
        li.appendChild(deleteBtn);

        return li;
    }

    function toggleTask(id) {
        tasks = tasks.map(task =>
            task.id === id ? { ...task, completed: !task.completed } : task
        );
        saveTasks();
        renderTasks();
    }

    function deleteTask(id) {
        tasks = tasks.filter(task => task.id !== id);
        saveTasks();
        renderTasks();
    }

    function saveTasks() {
        localStorage.setItem('tasks', JSON.stringify(tasks));
    }
});

Penjelasan Logika Penting:

  • createTaskElement: Fungsi ini membangun elemen <li> beserta isinya (checkbox, label, tombol hapus) untuk setiap tugas. Ini membuat kode lebih terorganisir.
  • toggleTask: Fungsi ini menggunakan .map() untuk membuat salinan array tasks, lalu mengubah properti completed pada tugas yang sesuai dengan ID. Ini adalah pendekatan yang aman (immutable) untuk memperbarui state.
  • deleteTask: Fungsi ini menggunakan .filter() untuk membuat array baru yang tidak menyertakan tugas dengan ID yang dihapus.
  • addEventListener pada elemen dinamis: Perhatikan bagaimana event listener untuk checkbox dan tombol hapus ditambahkan langsung saat elemen dibuat di dalam createTaskElement. Ini memastikan setiap elemen memiliki fungsionalitasnya masing-masing.

Tic-Tac-Toe: Logika JavaScript

Menulis logika permainan untuk mengatur giliran, mendeteksi pemenang, dan mengatur ulang game.

Kode Lengkap: script.js

Salin kode berikut dan simpan dalam file bernama script.js. Kode ini mengandung logika inti dari game Tic-Tac-Toe.

document.addEventListener('DOMContentLoaded', () => {
    const statusDisplay = document.getElementById('status-player');
    const cells = document.querySelectorAll('.cell');
    const restartButton = document.getElementById('restart-button');

    let gameActive = true;
    let currentPlayer = "X";
    let gameState = ["", "", "", "", "", "", "", "", ""];

    const winningMessage = () => `Pemain ${currentPlayer} Menang!`;
    const drawMessage = () => `Permainan Seri!`;
    const currentPlayerTurn = () => `Giliran: ${currentPlayer}`;

    statusDisplay.innerHTML = currentPlayerTurn();

    const winningConditions = [
        [0, 1, 2],
        [3, 4, 5],
        [6, 7, 8],
        [0, 3, 6],
        [1, 4, 7],
        [2, 5, 8],
        [0, 4, 8],
        [2, 4, 6]
    ];

    function handleCellPlayed(clickedCell, clickedCellIndex) {
        gameState[clickedCellIndex] = currentPlayer;
        clickedCell.innerHTML = currentPlayer;
        clickedCell.classList.add(currentPlayer.toLowerCase()); // Tambah kelas untuk styling
    }

    function handlePlayerChange() {
        currentPlayer = currentPlayer === "X" ? "O" : "X";
        statusDisplay.innerHTML = currentPlayerTurn();
    }

    function handleResultValidation() {
        let roundWon = false;
        for (let i = 0; i < winningConditions.length; i++) {
            const winCondition = winningConditions[i];
            const a = gameState[winCondition[0]];
            const b = gameState[winCondition[1]];
            const c = gameState[winCondition[2]];
            if (a === '' || b === '' || c === '') {
                continue;
            }
            if (a === b && b === c) {
                roundWon = true;
                break;
            }
        }

        if (roundWon) {
            statusDisplay.innerHTML = winningMessage();
            gameActive = false;
            return;
        }

        const roundDraw = !gameState.includes("");
        if (roundDraw) {
            statusDisplay.innerHTML = drawMessage();
            gameActive = false;
            return;
        }

        handlePlayerChange();
    }

    function handleCellClick(event) {
        const clickedCell = event.target;
        const clickedCellIndex = parseInt(clickedCell.getAttribute('data-cell-index'));

        if (gameState[clickedCellIndex] !== "" || !gameActive) {
            return;
        }

        handleCellPlayed(clickedCell, clickedCellIndex);
        handleResultValidation();
    }

    function handleRestartGame() {
        gameActive = true;
        currentPlayer = "X";
        gameState = ["", "", "", "", "", "", "", "", ""];
        statusDisplay.innerHTML = currentPlayerTurn();
        cells.forEach(cell => {
            cell.innerHTML = "";
            cell.classList.remove('x', 'o'); // Hapus kelas styling
        });
    }

    cells.forEach(cell => cell.addEventListener('click', handleCellClick));
    restartButton.addEventListener('click', handleRestartGame);
});

Penjelasan Logika Penting:

  • gameState: Array yang menyimpan status setiap sel (kosong, "X", atau "O"). Ini adalah "sumber kebenaran" (single source of truth) untuk kondisi permainan.
  • winningConditions: Array 2D yang berisi semua kombinasi indeks sel yang bisa membuat kemenangan (8 kemungkinan total).
  • handleResultValidation: Fungsi krusial yang dijalankan setelah setiap langkah. Ia akan memeriksa apakah ada kondisi kemenangan yang terpenuhi dengan membandingkan nilai di gameState dengan winningConditions.
  • handleCellClick: Fungsi ini menangani klik pengguna. Ia pertama-tama memeriksa apakah sel sudah terisi atau game sudah berakhir sebelum melanjutkan.
  • handleRestartGame: Fungsi ini mengatur ulang semua variabel state ke kondisi awal dan membersihkan tampilan papan, memungkinkan permainan dimulai lagi.

Belajar HTML

Pelajari fondasi web development dengan HTML, bahasa markup yang mendefinisikan struktur dan konten halaman web.

Apa itu HTML?

HTML (HyperText Markup Language) bukanlah bahasa pemrograman, melainkan bahasa markup yang digunakan untuk membuat struktur halaman web. HTML menggunakan tag untuk menandai elemen-elemen seperti heading, paragraf, link, gambar, dan lainnya.

Setiap halaman web yang kamu lihat di internet dibangun di atas struktur HTML. Browser seperti Chrome, Firefox, atau Safari membaca kode HTML ini lalu menampilkannya sebagai halaman web yang interaktif.

Struktur Dasar Dokumen HTML

Setiap dokumen HTML memiliki struktur dasar yang terdiri dari beberapa elemen wajib. Berikut adalah template paling sederhana:

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
</head>
<body>
    <!-- Konten halaman web ada di sini -->
    <h1>Ini adalah Heading</h1>
    <p>Ini adalah paragraf.</p>
</body>
</html>

Penjelasan Tag Penting:

  • <!DOCTYPE html>: Mendeklarasikan bahwa dokumen ini adalah HTML5.
  • <html>: Elemen root yang membungkus seluruh konten halaman.
  • <head>: Berisi informasi meta tentang halaman (seperti judul, charset, link ke CSS) yang tidak ditampilkan langsung di halaman.
  • <body>: Berisi semua konten yang terlihat oleh pengguna (teks, gambar, video, dll).

Tag-Tag HTML Umum

HTML memiliki banyak tag untuk berbagai jenis konten. Berikut adalah beberapa yang paling sering digunakan:

Heading & Paragraf
<h1> hingga <h6> untuk judul, <p> untuk paragraf.
Link & Gambar
<a href="..."> untuk link, <img src="..."> untuk gambar.
Daftar
<ul> (unordered), <ol> (ordered), <li> (item).
Pemformatan Teks
<strong> (penting), <em> (penekanan), <br> (baris baru).

Atribut HTML

Tag HTML bisa memiliki atribut yang memberikan informasi tambahan tentang elemen tersebut. Atribut ditulis dalam tag pembuka dengan format nama="nilai".

Contoh:

<a href="https://www.example.com" target="_blank">Kunjungi Example</a>

Di sini, href adalah atribut yang menentukan URL tujuan link, dan target="_blank" adalah atribut yang membuka link di tab baru.

Semantic HTML5

HTML5 memperkenalkan tag semantic yang memberikan makna lebih jelas pada struktur halaman. Ini sangat baik untuk SEO dan aksesibilitas.

<header> - Bagian atas halaman atau seksi.
<nav> - Bagian navigasi.
<main> - Konten utama halaman.
<article> - Konten yang berdiri sendiri.
<section> - Seksi tematik dalam dokumen.
<footer> - Bagian bawah halaman atau seksi.

Belajar CSS

Ubah tampilan HTML yang polos menjadi halaman web yang indah dan menarik dengan CSS (Cascading Style Sheets).

Apa itu CSS?

CSS adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan format dokumen HTML. Dengan CSS, kamu bisa mengontrol warna, font, jarak (spacing), layout, dan banyak aspek visual lainnya dari sebuah website.

Istilah "Cascading" mengacu pada aturan prioritas. Jika ada beberapa aturan CSS yang berlaku untuk satu elemen yang sama, browser akan menentukan aturan mana yang paling penting (cascade down) berdasarkan aturan spesifisitas.

Cara Menambahkan CSS ke HTML

Ada tiga cara untuk menambahkan CSS ke halaman HTML:

  1. Inline CSS: Menulis CSS langsung di atribut style sebuah tag HTML. Not recommended untuk proyek besar karena sulit dikelola. 
    <p style="color: blue; font-size: 16px;">Ini paragraf biru.</p>
  2. Internal CSS: Menulis CSS di dalam tag <style> di bagian <head> dokumen HTML. Cocok untuk halaman satu halaman (single-page). 
    <head>
    <style>
        p {
            color: blue;
            font-size: 16px;
        }
    </style>
</head>
  3. External CSS (Best Practice): Menulis CSS di file terpisah (misalnya style.css) lalu menautkannya dari HTML. Ini adalah cara yang paling baik dan terorganisir. 
    <!-- Di file HTML -->
<head>
    <link rel="stylesheet" href="style.css">
</head>

<!-- Di file style.css -->
p {
    color: blue;
    font-size: 16px;
}

Sintaks Dasar CSS

Aturan CSS terdiri dari dua bagian utama: Selector dan Declaration Block.

selector {
    property: value;
    property: value;
}
  • Selector: Menunjuk ke elemen HTML yang ingin di-styling (misalnya p, h1, .my-class).
  • Property: Atribut gaya yang ingin diubah (misalnya color, font-size).
  • Value: Nilai yang diberikan pada property (misalnya blue, 16px).

Selector CSS

Selector adalah kunci untuk menargetkan elemen yang tepat. Berikut adalah selector yang paling umum:

p - Selector elemen (memilih semua <p>).
.my-class - Selector class (memilih semua elemen dengan class="my-class").
#my-id - Selector ID (memilih satu elemen dengan id="my-id").
p, h1 - Grouping selector (memilih semua <p> dan <h1>).
div p - Descendant selector (memilih semua <p> di dalam <div>).

Box Model

Setiap elemen HTML pada dasarnya adalah sebuah kotak. Box Model menjelaskan bagaimana kotak ini dibangun. Dari dalam ke luar, urutannya adalah:

  1. Content: Konten sebenarnya (teks, gambar).
  2. Padding: Ruang kosong di antara konten dan border.
  3. Border: Garis yang mengelilingi padding.
  4. Margin: Ruang kosong di luar border, memisahkan elemen dari elemen lain.

Properti box-sizing: border-box; sangat berguna untuk membuat perhitungan ukuran elemen menjadi lebih intuitif.

Layout dengan Flexbox dan Grid

CSS modern menyediakan dua metode layout yang sangat kuat:

Flexbox

Cocok untuk layout satu dimensi (baris atau kolom). Sangat efektif untuk menyusun item, membuat navigasi, atau menengahkan konten.

display: flex;
Grid

Cocok untuk layout dua dimensi (baris dan kolom). Sempurna untuk membuat keseluruhan layout halaman, seperti yang kita lihat di website modern.

display: grid;

Responsive Design

Website harus terlihat baik di berbagai ukuran layar (desktop, tablet, mobile). CSS Media Queries memungkinkan kita menerapkan gaya yang berbeda berdasarkan karakteristik perangkat.

/* Gaya default untuk desktop */
.container {
    width: 1200px;
}

/* Gaya untuk tablet dan layar lebih kecil */
@media (max-width: 768px) {
    .container {
        width: 100%;
        padding: 10px;
    }
}

Belajar JavaScript

Bawa halaman webmu hidup dengan JavaScript, bahasa pemrograman yang menggerakkan interaktivitas dan logika di browser.

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman tingkat tinggi yang terutama digunakan untuk membuat halaman web interaktif. Di dalam browser, JavaScript bisa memanipulasi DOM (Document Object Model), merespon event pengguna (klik, ketik, scroll), dan berkomunikasi dengan server.

Berbeda dengan HTML (struktur) dan CSS (gaya), JavaScript adalah "otak" yang memungkinkan logika dan dinamisme pada halaman web. Saat ini, JavaScript juga bisa digunakan di sisi server (Node.js) dan untuk pengembangan aplikasi mobile (React Native).

Menambahkan JavaScript ke HTML

Sama seperti CSS, ada cara untuk menambahkan JavaScript, tetapi yang paling umum dan direkomendasikan adalah file eksternal.

<!-- Di file HTML, biasanya di akhir <body> -->
<script src="script.js"></script>

Menaruh tag <script> di akhir <body> adalah praktik terbaik karena memastikan semua elemen HTML telah dimuat sebelum JavaScript mencoba memanipulasinya.

Variabel dan Tipe Data

Variabel adalah wadah untuk menyimpan data. Di JavaScript modern, gunakan let dan const.

// Variabel yang bisa diubah nilainya
let age = 25;
age = 26; // OK

// Variabel yang tidak bisa diubah nilainya (konstanta)
const name = "Alice";
// name = "Bob"; // Akan menyebabkan error

// Tipe Data Dasar
let myNumber = 10;           // Number
let myString = "Hello";       // String
let isTrue = true;            // Boolean
let nothing = null;           // Null (sengaja kosong)
let notDefined;               // Undefined (belum diberi nilai)

Array dan Object

Ini adalah dua struktur data yang paling penting di JavaScript.

Array (Daftar)
let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]); // "Apple"
fruits.push("Orange"); // Menambah item
Object (Struktur)
let person = {
    name: "Bob",
    age: 30,
    isStudent: false
};
console.log(person.name); // "Bob"

Fungsi (Function)

Fungsi adalah blok kode yang bisa digunakan kembali. Mereka menerima input (parameter) dan bisa mengembalikan output (return value).

// Deklarasi fungsi
function greet(name) {
    return "Hello, " + name;
}

// Arrow function (lebih modern)
const add = (a, b) => a + b;

// Memanggil fungsi
console.log(greet("World")); // "Hello, World"
console.log(add(5, 3));      // 8

Manipulasi DOM (Document Object Model)

Ini adalah inti dari interaktivitas web. DOM adalah representasi struktur halaman HTML yang bisa diakses dan diubah oleh JavaScript.

allButtons.forEach(button => {
    button.addEventListener('click', () => {
        alert('Tombol diklik!');

Event dan Event Listener

Event adalah aksi yang terjadi di halaman (klik mouse, tekan keyboard, dll). Event Listener adalah fungsi yang "menunggu" event tertentu terjadi lalu menjalankan kode.

Contoh yang paling umum adalah DOMContentLoaded, yang memastikan kode JavaScript berjalan hanya setelah seluruh HTML dimuat, dan 'click', yang merespons klik pengguna.

        console.error('Gagal mengambil data:', error);

Belajar React

Bangun antarmuka pengguna (UI) yang kompleks dan efisien dengan React, library JavaScript paling populer dari Facebook.

Apa itu React?

React adalah library JavaScript untuk membangun User Interface (UI). Dikembangkan oleh Facebook, React memungkinkan kita membuat UI yang interaktif dengan memecahnya menjadi komponen-komponen kecil yang bisa digunakan kembali.

Konsep kunci React adalah declarative. Alih-alih memberitahu browser "cara" mengubah UI (imperative), kita hanya perlu mendeskripsikan "bagaimana seharusnya" UI terlihat untuk setiap state, dan React akan menangani pembaruan DOM secara efisien.

Konsep Inti React

1. Komponen (Components)

Komponen adalah blok bangunan utama React. Mereka seperti fungsi JavaScript yang menerima input (props) dan mengembalikan elemen React yang mendeskripsikan apa yang harus ditampilkan di layar.

// Komponen Fungsi
function Welcome(props) {
    return <h1>Halo, {props.name}</h1>;
}

// Menggunakan komponen
<Welcome name="Alice" />
2. JSX (JavaScript XML)

JSX adalah ekstensi sintaks untuk JavaScript yang memungkinkan kita menulis kode yang mirip HTML langsung di dalam file JavaScript. Ini membuat kode komponen lebih intuitif.

const element = <h1>Halo, dunia!</h1>;
3. Props (Properties)

Props adalah argumen yang dilewatkan ke komponen, mirip seperti parameter fungsi. Mereka bersifat read-only dan digunakan untuk mengirim data dari komponen induk ke anak.

4. State

State adalah objek yang menyimpan data yang bisa berubah seiring waktu di dalam komponen. Ketika state berubah, React akan merender ulang komponen untuk mencerminkan perubahan tersebut. Hook useState digunakan untuk menambah state ke komponen fungsi.

import { useState } from 'react';

function Counter() {
    const [count, setCount] = useState(0);

    return (
        <div>
            <p>Anda menekan {count} kali</p>
            <button onClick={() => setCount(count + 1)}>
                Klik saya
            </button>
        </div>
    );
}

React Hooks

Hooks adalah fungsi khusus yang memungkinkan kita "mengaitkan" state dan fitur React lainnya ke komponen fungsi. Mereka membuat kode lebih ringkas dan mudah dipahami.

useState

Seperti yang sudah dibahas, ini adalah Hook yang paling dasar untuk menambah state lokal ke komponen.

useEffect

Digunakan untuk melakukan "side effect" seperti mengambil data, berlangganan, atau secara manual mengubah DOM. Ia berjalan setelah setiap render.

Manajemen State Lanjutan (Context API)

Untuk aplikasi yang lebih besar, meneruskan state melalui banyak tingkatan komponen (prop drilling) bisa menjadi masalah. React Context API menyediakan cara untuk berbagi data (state) ke seluruh komponen tanpa harus melewatkan props secara manual di setiap level.

Ekosistem React

React tidak berdiri sendiri. Ia didukung oleh ekosistem yang kaya:

  • React Router: Untuk menangani navigasi dan routing di aplikasi single-page.
  • Axios / Fetch API: Untuk melakukan request HTTP dan berkomunikasi dengan backend API.
  • Redux / Zustand: Library manajemen state global untuk aplikasi yang sangat kompleks.
  • Next.js: Framework React yang menawarkan rendering di sisi server (SSR) dan generasi situs statis (SSG).

Belajar Python

Jelajahi kekuatan Python, bahasa pemrograman serbaguna yang sangat populer untuk web development, data science, automasi, dan kecerdasan buatan.

Apa itu Python?

Python adalah bahasa pemrograman tingkat tinggi yang diketahui karena sintaksnya yang bersih, mudah dibaca, dan sederhana. Filosofi "Zen of Python" menekankan pada keterbacaan dan kesederhanaan.

Python adalah bahasa yang diinterpretasikan, yang berarti kode bisa dijalankan baris per baris, membuat proses development dan debugging lebih cepat. Ia sangat serbaguna dan digunakan di berbagai bidang, dari pengembangan web hingga penelitian ilmiah.

Instalasi dan Menjalankan Python

Untuk memulai, unduh dan instal Python dari situs resmi python.org. Pastikan untuk mencentang "Add Python to PATH" saat instalasi.

Kamu bisa menjalankan kode Python dengan dua cara:

  1. Interactive Mode: Buka terminal/command prompt, ketik python, dan tulis kode langsung di sana.
  2. Script Mode: Simpan kode dalam file dengan ekstensi .py (misalnya app.py), lalu jalankan dari terminal dengan perintah python app.py.

Sintaks Dasar Python

Python menggunakan indentasi (spasi atau tab) untuk mendefinisikan blok kode, bukan tanda kurung kurawal seperti di JavaScript atau C++.

# Ini adalah komentar
# Variabel tidak perlu deklarasi tipe
name = "Python"
version = 3.9
is_fun = True

# Print output
print(f"Selamat datang di {name} versi {version}")

# Struktur kontrol (if/else)
if is_fun:
    print("Python itu menyenangkan!")
else:
    print("Mungkin kamu perlu coba lagi.")

# Perulangan (for loop)
for i in range(5):
    print(f"Perulangan ke-{i}")

Struktur Data: List, Tuple, Dictionary

Python memiliki struktur data bawaan yang sangat powerful.

List (Array)

Koleksi item yang terurut dan bisa diubah (mutable).

fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits[0]) # "apple"
Tuple

Koleksi item yang terurut tetapi tidak bisa diubah (immutable).

coordinates = (10, 20)
# coordinates[0] = 15 # Akan error
Dictionary (Object)

Koleksi pasangan key-value yang tidak terurut.

person = {
    "name": "Budi",
    "age": 30
}
print(person["name"]) # "Budi"

Fungsi dan Modul

Fungsi didefinisikan dengan kata kunci def. Python memiliki standar library yang sangat besar (modul) yang bisa langsung digunakan.

# Mendefinisikan fungsi
def greet(name, language="Indonesia"):
    if language == "Indonesia":
        return f"Halo, {name}!"
    else:
        return f"Hello, {name}!"

# Menggunakan fungsi
print(greet("Andi")) # "Halo, Andi!"

# Mengimpor modul
import math
print(math.sqrt(16)) # 4.0

# Mengimpor fungsi spesifik dari modul
from datetime import datetime
print(datetime.now())

Python untuk Web Development

Meskipun bukan pilihan utama untuk frontend, Python sangat kuat di sisi backend. Framework web populer seperti Django dan Flask memungkinkan kita membangun API dan aplikasi web yang robust dengan cepat.

Flask adalah micro-framework yang ringan dan fleksibel, sementara Django adalah "batteries-included" framework yang menyediakan hampir semua yang dibutuhkan untuk membangun aplikasi web kompleks.

Python untuk Data Science & AI

Ini adalah area di mana Python benar-benar bersinar. Library seperti Pandas (untuk manipulasi data), NumPy (untuk komputasi numerik), Matplotlib (untuk visualisasi), dan Scikit-learn / TensorFlow (untuk machine learning) adalah standar industri.
