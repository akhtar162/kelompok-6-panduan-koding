Jejak Evolusi Koding

Menelusuri perjalanan dari konsep mekanis di abad ke-19 hingga era kecerdasan buatan modern yang kita nikmati hari ini.

Kembali ke Halaman Utama

1. Awal Mula (Abad ke-19)

Charles Babbage (1830-an): Matematikawan Inggris ini merancang Mesin Analitik (Analytical Engine), sebuah konsep mesin komputasi mekanis serbaguna yang dianggap sebagai prototipe komputer modern pertama.

Ada Lovelace (1840-an): Dikenal sebagai programmer komputer pertama, ia menulis algoritma untuk Mesin Analitik untuk menghitung bilangan Bernoulli dan menyadari potensi mesin di luar kalkulasi angka.

2. Era Komputer Awal (1940-an - 1950-an)

Perang Dunia II mempercepat pengembangan komputer elektronik seperti ENIAC. Pemrograman awalnya dilakukan dengan pengkabelan ulang manual atau kode mesin (biner). Bahasa Assembly muncul pada 1949, diikuti oleh bahasa tingkat tinggi pertama seperti Autocode (1952) dan FORTRAN (1957) yang merevolusi cara manusia berinteraksi dengan mesin.

3. Perkembangan & Paradigma Baru (1960-an - 1980-an)

Bahasa berpengaruh seperti ALGOL, BASIC, dan PASCAL lahir. Namun, Bahasa C (1972) oleh Dennis Ritchie menjadi tonggak sejarah karena keseimbangannya, memengaruhi C++, Java, dan Python. Konsep Pemrograman Berorientasi Objek (OOP) juga menjadi populer, memungkinkan kode diatur menjadi objek yang lebih terstruktur.

4. Era Internet & Modern (1990-an - Sekarang)

Ledakan internet memicu lahirnya bahasa untuk pengembangan web: HTML (1991) untuk struktur, Python (1991) sebagai bahasa serbaguna, serta Java (1995) dan JavaScript (1995) yang krusial untuk fungsionalitas web. Abad ke-21 menyaksikan revolusi komputasi mobile, cloud, Big Data, dan AI, dengan kerangka kerja (framework) modern menjadi standar untuk menyederhanakan pengembangan.

Notepad: Struktur HTML

Membangun kerangka dasar aplikasi notepad dengan elemen form dan wadah untuk catatan.

Kode Lengkap: index.html

Salin kode berikut dan simpan dalam file bernama index.html. Struktur ini mencakup header, formulir untuk membuat catatan baru, dan area untuk menampilkan daftar catatan.

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Aplikasi Notepad</title>
    <!-- CSS akan ditautkan di sini -->
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="container">
        <header>
            <h1>Aplikasi Notepad Sederhana</h1>
        </header>
        
        <main>
            <section class="note-form">
                <h2>Buat Catatan Baru</h2>
                <!-- Form untuk input judul dan isi catatan -->
                <form id="note-form">
                    <div class="form-group">
                        <label for="note-title">Judul Catatan</label>
                        <input type="text" id="note-title" placeholder="Masukkan judul catatan" required>
                    </div>
                    <div class="form-group">
                        <label for="note-content">Isi Catatan</label>
                        <textarea id="note-content" placeholder="Tulis catatan Anda di sini" required></textarea>
                    </div>
                    <button type="submit" class="btn">Simpan Catatan</button>
                </form>
            </section>
            
            <section class="notes-list">
                <h2>Daftar Catatan</h2>
                <!-- Wadah untuk menampilkan catatan yang sudah dibuat -->
                <div id="notes-container">
                    <p class="empty-state">Belum ada catatan. Buat catatan baru!</p>
                </div>
            </section>
        </main>
    </div>
    
    <!-- JavaScript akan ditautkan di sini -->
    <script src="script.js"></script>
</body>
</html>

Penjelasan Elemen Penting:

  • <form id="note-form">: Formulir yang akan menangani input pengguna. Atribut id penting agar bisa diakses oleh JavaScript.
  • <label for="...">: Label untuk meningkatkan aksesibilitas dan keterhubungan dengan input.
  • <input type="text" id="note-title">: Input teks satu baris untuk judul. id digunakan JavaScript untuk mengambil nilainya.
  • <textarea id="note-content">: Area teks multi-baris untuk isi catatan.
  • <button type="submit">: Tombol untuk mengirimkan formulir.
  • <div id="notes-container">: Wadah kosong yang akan diisi oleh catatan-catatan secara dinamis menggunakan JavaScript.

Notepad: Logika JavaScript

Menghidupkan aplikasi notepad dengan fungsi untuk menambah, menampilkan, dan menghapus catatan, serta menyimpannya di browser.

Kode Lengkap: script.js

Salin kode berikut dan simpan dalam file bernama script.js. Kode ini mengatur semua interaktivitas aplikasi.

// Tunggu hingga seluruh konten HTML dimuat
document.addEventListener('DOMContentLoaded', function() {
    // Mendapatkan elemen-elemen yang diperlukan dari DOM
    const noteForm = document.getElementById('note-form');
    const noteTitle = document.getElementById('note-title');
    const noteContent = document.getElementById('note-content');
    const notesContainer = document.getElementById('notes-container');
    
    // Mengambil catatan yang tersimpan di localStorage, atau buat array kosong jika tidak ada
    let notes = JSON.parse(localStorage.getItem('notes')) || [];
    
    // Menampilkan catatan yang ada saat halaman dimuat
    displayNotes();
    
    // Event listener untuk saat formulir disubmit
    noteForm.addEventListener('submit', function(e) {
        // Mencegah perilaku default form (reload halaman)
        e.preventDefault();
        
        // Membuat objek catatan baru
        const newNote = {
            id: Date.now(), // ID unik menggunakan timestamp
            title: noteTitle.value,
            content: noteContent.value,
            createdAt: new Date().toLocaleString()
        };
        
        // Menambahkan catatan baru ke array 'notes'
        notes.push(newNote);
        
        // Menyimpan array 'notes' ke localStorage
        saveNotes();
        
        // Menampilkan ulang daftar catatan
        displayNotes();
        
        // Mengosongkan form
        noteForm.reset();
    });
    
    // Fungsi untuk menyimpan catatan ke localStorage
    function saveNotes() {
        localStorage.setItem('notes', JSON.stringify(notes));
    }
    
    // Fungsi untuk menampilkan semua catatan
    function displayNotes() {
        // Mengosongkan container
        notesContainer.innerHTML = '';
        
        // Jika tidak ada catatan, tampilkan pesan
        if (notes.length === 0) {
            notesContainer.innerHTML = '
Belum ada catatan. Buat catatan baru!
';
            return;
        }
        
        // Menampilkan setiap catatan
        notes.forEach(note => {
            const noteElement = createNoteElement(note);
            notesContainer.appendChild(noteElement);
        });
    }
    
    // Fungsi untuk membuat elemen catatan
    function createNoteElement(note) {
        // Membuat elemen div untuk catatan
        const noteDiv = document.createElement('div');
        noteDiv.className = 'note-item';
        noteDiv.dataset.id = note.id;
        
        // Membuat elemen untuk judul
        const titleElement = document.createElement('h3');
        titleElement.className = 'note-title';
        titleElement.textContent = note.title;
        
        // Membuat elemen untuk konten
        const contentElement = document.createElement('p');
        contentElement.className = 'note-content';
        contentElement.textContent = note.content;
        
        // Membuat elemen untuk tanggal
        const dateElement = document.createElement('small');
        dateElement.className = 'note-date';
        dateElement.textContent = note.createdAt;
        
        // Membuat tombol hapus
        const deleteButton = document.createElement('button');
        deleteButton.className = 'delete-btn';
        deleteButton.textContent = 'Hapus';
        deleteButton.addEventListener('click', function() {
            deleteNote(note.id);
        });
        
        // Membuat container untuk tombol
        const actionsDiv = document.createElement('div');
        actionsDiv.className = 'note-actions';
        actionsDiv.appendChild(deleteButton);
        
        // Menambahkan semua elemen ke div catatan
        noteDiv.appendChild(titleElement);
        noteDiv.appendChild(contentElement);
        noteDiv.appendChild(dateElement);
        noteDiv.appendChild(actionsDiv);
        
        return noteDiv;
    }
    
    // Fungsi untuk menghapus catatan
    function deleteNote(id) {
        // Filter array untuk menghapus catatan dengan ID yang sesuai
        notes = notes.filter(note => note.id !== id);
        
        // Menyimpan perubahan ke localStorage
        saveNotes();
        
        // Menampilkan ulang daftar catatan
        displayNotes();
    }
});

Penjelasan Kode Penting:

  • localStorage: API browser untuk menyimpan data secara lokal di perangkat pengguna.
  • JSON.parse() dan JSON.stringify(): Mengkonversi array objek ke string dan sebaliknya untuk penyimpanan di localStorage.
  • addEventListener('submit', ...): Menangani pengiriman form tanpa reload halaman.
  • createElement(): Membuat elemen HTML secara dinamis dengan JavaScript.
  • dataset.id: Menyimpan ID catatan di atribut data untuk akses mudah.

Pro Tip

Untuk membuat aplikasi lebih canggih, Anda bisa menambahkan fitur "Edit". Saat tombol "Edit" diklik, Anda bisa mengubah elemen catatan menjadi <input> dan <textarea> lagi, lalu menyimpan perubahannya. Ini akan melatih Anda lebih dalam tentang manipulasi DOM dan state management.

To-Do List: Logika JavaScript

Menghidupkan aplikasi dengan fungsi untuk menambah, menyelesaikan, menghapus, dan menyimpan tugas.

Kode Lengkap: script.js

Buat file script.js dan salin kode ini. Kode ini akan menangani semua interaksi pengguna dan manajemen data tugas.

document.addEventListener('DOMContentLoaded', () => {
    const form = document.getElementById('todo-form');
    const input = document.getElementById('todo-input');
    const list = document.getElementById('todo-list');

    // Muat tugas dari localStorage saat halaman dibuka
    const savedTodos = JSON.parse(localStorage.getItem('todos')) || [];
    savedTodos.forEach(todo => addTodoItem(todo.text, todo.completed));

    form.addEventListener('submit', (e) => {
        e.preventDefault();
        const text = input.value.trim();
        if (text !== '') {
            addTodoItem(text, false);
            saveTodos();
            input.value = '';
        }
    });

    function addTodoItem(text, completed) {
        const li = document.createElement('li');
        li.className = 'todo-item';
        if (completed) {
            li.classList.add('completed');
        }

        const span = document.createElement('span');
        span.textContent = text;
        span.addEventListener('click', () => {
            li.classList.toggle('completed');
            saveTodos();
        });

        const deleteBtn = document.createElement('button');
        deleteBtn.className = 'delete-btn';
        deleteBtn.textContent = 'X';
        deleteBtn.addEventListener('click', () => {
            li.remove();
            saveTodos();
        });

        li.appendChild(span);
        li.appendChild(deleteBtn);
        list.appendChild(li);
    }

    function saveTodos() {
        const todos = [];
        document.querySelectorAll('.todo-item').forEach(item => {
            todos.push({
                text: item.querySelector('span').textContent,
                completed: item.classList.contains('completed')
            });
        });
        localStorage.setItem('todos', JSON.stringify(todos));
    }
});

Penjelasan Kode Penting:

  • addEventListener('DOMContentLoaded', ...): Memastikan kode JavaScript berjalan setelah seluruh halaman HTML dimuat.
  • e.preventDefault(): Mencegah form mengirim data dan me-reload halaman.
  • addTodoItem(): Fungsi utama yang membuat elemen-elemen HTML (li, span, button) untuk setiap tugas baru.
  • classList.toggle('completed'): Cara mudah untuk menambah/menghapus kelas CSS untuk menandai tugas selesai.
  • localStorage.setItem() dan localStorage.getItem(): Digunakan untuk menyimpan dan mengambil data tugas sehingga tidak hilang saat halaman ditutup.
  • JSON.stringify() dan JSON.parse(): Mengubah data objek/array menjadi string untuk disimpan di localStorage, dan sebaliknya.

Pro Tip

Daripada menyimpan seluruh elemen HTML di localStorage, lebih efisien menyimpan datanya saja (teks dan status selesai) sebagai array of objects. Ini membuat data lebih bersih, mudah dikelola, dan lebih cepat saat disimpan atau dimuat.

Tic-Tac-Toe: Logika JavaScript

Mengimplementasikan aturan permainan, pergantian giliran, pengecekan pemenang, dan fungsi reset.

Kode Lengkap: script.js

Buat file script.js dan salin kode ini. Ini adalah "otak" dari permainan Tic-Tac-Toe kita.

document.addEventListener('DOMContentLoaded', () => {
    const cells = document.querySelectorAll('.cell');
    const statusText = document.getElementById('status');
    const restartButton = document.getElementById('restart-button');

    let currentPlayer = 'X';
    let gameState = ["", "", "", "", "", "", "", "", ""];
    let gameActive = true;

    const winningConditions = [
        [0, 1, 2], [3, 4, 5], [6, 7, 8], // Baris
        [0, 3, 6], [1, 4, 7], [2, 5, 8], // Kolom
        [0, 4, 8], [2, 4, 6]             // Diagonal
    ];

    function handleCellClick(e) {
        const clickedCell = e.target;
        const clickedCellIndex = parseInt(clickedCell.getAttribute('data-index'));

        if (gameState[clickedCellIndex] !== "" || !gameActive) {
            return;
        }

        gameState[clickedCellIndex] = currentPlayer;
        clickedCell.textContent = currentPlayer;
        clickedCell.classList.add(currentPlayer.toLowerCase());

        checkResult();
    }

    function checkResult() {
        let roundWon = false;
        for (let i = 0; i < winningConditions.length; i++) {
            const winCondition = winningConditions[i];
            let a = gameState[winCondition[0]];
            let b = gameState[winCondition[1]];
            let c = gameState[winCondition[2]];
            if (a === '' || b === '' || c === '') {
                continue;
            }
            if (a === b && b === c) {
                roundWon = true;
                break;
            }
        }

        if (roundWon) {
            statusText.textContent = `Pemain ${currentPlayer} Menang!`;
            gameActive = false;
            return;
        }

        if (!gameState.includes("")) {
            statusText.textContent = `Permainan Seri!`;
            gameActive = false;
            return;
        }

        currentPlayer = currentPlayer === 'X' ? 'O' : 'X';
        statusText.textContent = `Giliran ${currentPlayer}`;
    }

    function restartGame() {
        currentPlayer = 'X';
        gameState = ["", "", "", "", "", "", "", "", ""];
        gameActive = true;
        statusText.textContent = `Giliran ${currentPlayer}`;
        cells.forEach(cell => {
            cell.textContent = '';
            cell.classList.remove('x', 'o');
        });
    }

    cells.forEach(cell => cell.addEventListener('click', handleCellClick));
    restartButton.addEventListener('click', restartGame);
});

Penjelasan Kode Penting:

  • gameState: Sebuah array yang merepresentasikan 9 sel di papan. Ini adalah "single source of truth" untuk kondisi permainan.
  • winningConditions: Array 2D yang berisi semua kemungkinan kombinasi kemenangan.
  • handleCellClick(): Fungsi yang dipanggil saat sel diklik. Fungsi ini memeriksa apakah sel valid untuk diisi, lalu memperbarui gameState dan UI.
  • checkResult(): Setelah setiap langkah, fungsi ini memeriksa apakah ada pemenang atau permainan seri dengan cara mengiterasi winningConditions.
  • currentPlayer = currentPlayer === 'X' ? 'O' : 'X';: Ekspresi ternary yang elegan untuk mengganti giliran pemain.
  • restartGame(): Mengembalikan semua variabel state ke nilai awal dan membersihkan papan.

Pro Tip

Untuk membuat AI (lawan komputer) sederhana, Anda bisa membuat fungsi aiMove(). Setelah pemain bergerak, panggil fungsi ini. Di dalamnya, cari sel kosong secara acak dan isi dengan 'O'. Untuk AI yang lebih pintar, coba implementasikan logika untuk memblokir kemenangan pemain atau mencari langkah kemenangan untuk AI.
